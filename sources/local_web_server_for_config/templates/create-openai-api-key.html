Description et utilité d'une clé API

Une clé API fonctionne comme un identifiant lié à votre compte permettant d'utiliser un service en ligne. Tars utilise le modèle ChatGPT de l'entreprise OpenAI pour générer ses réponses, et son usage via du code nécessite votre authentification; c'est pourquoi nous avons besoin d'utiliser votre clé API, afin de l'indiquer. Pour des raisons de confidentialité, nous ne stockons votre clé api qu'exclusivement sur votre appareil.

Vous êtes :

  • Sur ordinateur
  • Sur téléphone ou tablette

Sur ordinateur

Se connecter à votre compte

C'est sur cette page que vous pourrez obtenir votre clé d'API, une fois connecté.e.

Panneau de configuration des clés API

Vous voici sur l'onglet de votre compte où vous pouvez générer la clé API dont Tars a besoin pour répondre à vos questions. Cliquez sur "+ Create new secret key".

Créer la clé API

Dans le panneau apparut, vous devez donner un nom à votre clé (ce dernier n'aura pas de particulière importance), et s'assurer que la partie "Permissions" soit sur All. Une fois fait, cliquez sur "Create secret key".

La copier et l'indiquer à Tars

Il ne vous reste plus qu'à copier le code qui vous est donné (la fameuse clé API) et la coller dans le champ de texte de la page précédente pour la tester.
