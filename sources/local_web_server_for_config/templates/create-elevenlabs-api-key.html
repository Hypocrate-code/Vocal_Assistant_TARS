Pourquoi utiliser ElevenLabs ?

ElevenLabs permet de générer une multitude de voix humanoïdes. Configurer son utilisation pourra améliorer votre expérience, mais avec une contrepartie : le temps de réponse sera légérement plus long, car Elevenlabs est un service sur le web (une API), contrairement aux voix robotiques préconfigurées de Tars.

Vous êtes :

  • Sur ordinateur
  • Sur téléphone ou tablette

Sur ordinateur

Se rendre sur la page web d'ElevenLabs

C'est sur le site d'ElevenLabs que vous pourrez obtenir votre clé d'API, une fois connecté.e. Cette dernière représentera votre compte lorsque Tars essaiera de se connecter à ce service. Cet identifiant est unique, confidentiel et nous ne le partageons pas.

Se connecter ou créer un compte

Votre compte ElevenLabs permettra à Tars de générer gratuitement 10000 caractères avec la voix que vous sélectionnerez. Une fois ce quota passé, vous devrez : soit payer pour continuer d'utiliser ce service, soit utiliser un autre compte lié à un numéro de téléphone différent.

Naviguer jusqu'à votre profil

Cliquez sur l'icone de votre compte, puis sur "Profile + API key" dans le menu déroulant.

Copier votre clé API

Voici votre clé API pour ElevenLabs ; copiez-la et collez-la sur la page précédente pour tester votre connexion.
